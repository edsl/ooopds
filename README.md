# Ooopds

Ooopds it's an [OPDS][] generator, derived from [rAtom][].

## Installation

Add this line to your application's Gemfile:

    gem 'ooopds'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ooopds

## Usage

You can find more about OPDS [here](http://discover-opds.org/).
